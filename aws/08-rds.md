# Day22: RDS（MySQL）

## ■目的
RDS（MySQL）を構築し、EC2から接続できるようにする。

---

## ■実施内容

### ① RDS作成

- エンジン：MySQL
- テンプレート：無料利用枠
- DBインスタンス識別子：my-rds
- マスターユーザー：admin
- インスタンスサイズ：db.t3.micro
- ストレージ：20GB

---

### ② 接続設定

- VPC：EC2と同じ
- パブリックアクセス：有効（検証用）

---

### ③ セキュリティグループ設定

RDSのセキュリティグループに以下を追加：

    タイプ：MYSQL/Aurora
    ポート：3306
    ソース：EC2のセキュリティグループ

※IPではなくセキュリティグループを指定することで、より安全に接続制御できる

---

### ④ MySQLクライアントインストール（EC2）

    sudo apt update
    sudo apt install mysql-client -y

---

### ⑤ RDSへ接続

    mysql -h <RDSエンドポイント> -u admin -p

パスワード入力後、接続成功

---

### ⑥ 動作確認

    show databases;

---

## ■学んだこと

- RDSはマネージドなリレーショナルデータベースサービス
- EC2からRDSへ接続するにはセキュリティグループの設定が必要
- 接続元はIPではなくセキュリティグループで制御するのがベストプラクティス
- 同一VPC内で構成する必要がある
