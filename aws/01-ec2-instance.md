# Day15 EC2 インスタンス作成

## 環境

Cloud : AWS  
Service : EC2  
Instance Name : aws-study  
OS : Ubuntu Server 22.04 LTS  

## 作業内容

EC2インスタンスを作成。

Instance type  
t2.micro  

Key pair  
aws-study-key.pem  

Public IP が自動割り当てされる設定で起動。

## 学び

EC2はAWS上で仮想サーバーを起動できるサービス。  
クラウド上でLinuxサーバーを簡単に作成できる。  
SSH接続にはキーペアが必要。
